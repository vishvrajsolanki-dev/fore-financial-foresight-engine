# FORE Blueprint 2 — Superseded

**This document is superseded by [FORE_Blueprint.md](./FORE_Blueprint.md).**

The original FORE Blueprint 2 (RupeeIQ v2 rescoped → 8-hour build spec) proposed JWT auth, PostgreSQL via Prisma, and a split Express/FastAPI backend. Those items were cut during the brutal review documented in the blueprint upgrade — see [ROADMAP.md](./ROADMAP.md) Tier 3 for the reasoning. The live hackathon build uses Next.js + an always-on Python ML service on Render, client-side `financial_context`, and no auth or database.

For the current product spec, demo script, and feature set, use **FORE_Blueprint.md** only.
